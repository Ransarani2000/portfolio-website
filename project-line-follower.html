Line Following Robot

IR sensor-based autonomous navigation (Arduino)

Code (optional) Demo (optional)
Fully assembled robot prototype

Project Overview

Built as a foundational robotics project to practice sensor-based control and embedded programming. The robot uses an IR sensor array to detect a black line and adjusts motor speeds through a motor driver to maintain stable tracking across curves and sharp turns.

Technologies Used

Arduino Uno IR Sensors L298N Motor Driver Embedded C++ PID (optional)

Key Features

  • Sensor calibration for changing ambient light
  • Stable cornering and recovery behavior
  • Speed control for smoother tracking

Challenges & Approach

Ambient light variation: Added a startup calibration routine to set dynamic thresholds.

Sharp turns: Improved control logic (optionally PID) to reduce oscillation and regain the line quickly.

Results

Successfully followed complex paths including curves and 90° turns. This project established a strong base for later autonomous robotics work.
