Project Overview

This IoT home automation system allows users to control lighting and a door lock through both physical switches and a smartphone over Wi‑Fi. The ESP32 hosts a lightweight HTTP web interface and drives a relay module for AC loads while controlling a servo mechanism for the lock.

Technologies Used

ESP32 Embedded C++ Arduino IDE HTTP Web Server Relay Module Servo Control KiCad

Key Features

  • Manual override switches (works even without phone)
  • Wi‑Fi control via browser UI
  • AC relay isolation + safer wiring layout
  • Auto-reconnect and watchdog logic for reliability

Challenges & Approach

Wi‑Fi stability: Implemented auto-reconnect logic and a watchdog timer to keep the controller responsive.

Safe AC switching: Used opto-isolated relays and separated HV/LV areas in the layout to reduce noise and improve safety.

Results

Achieved responsive local-network control from any phone/PC and reliable operation during daily use. Strengthened skills in embedded networking, safe interfacing, and PCB design.
