Autonomous Cleaning Robot

Complete mechatronic design: CAD, PCB, embedded control, and sensor-based navigation

3D mechanical design in SolidWorks
Custom PCB designed in Proteus
Navigation test

Project Overview

Designed and built an autonomous indoor floor-cleaning robot prototype including mechanical chassis design, a custom control PCB, embedded firmware, and sensor-based obstacle detection. The robot uses ToF sensors for distance measurement and stepper motor control for accurate movement.

Technologies Used

SolidWorks Proteus PCB ESP32 VL53L0X ToF Stepper Drivers Embedded C++

Key Features

  • Modular chassis for rapid iteration
  • Power distribution with proper regulation
  • Obstacle avoidance using ToF sensors
  • System integration: CAD → PCB → firmware

Challenges & Approach

Mechanical integration: Iterated mounting and routing to reduce vibration and improve sensor placement.

Power management: Implemented regulated rails for MCU, sensors, and drivers with decoupling and safe grounding.

Navigation logic: Built a simple rule-based strategy for obstacle avoidance and coverage.

Results

Achieved reliable obstacle detection and systematic movement in a room-scale environment. Strengthened full-stack mechatronics skills across design, electronics, and firmware.
