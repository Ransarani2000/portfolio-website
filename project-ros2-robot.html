Professional Development

ROS 2: Map, Localization & Navigation ROS 2: Odometry & Control Systems
Prototype with Raspberry Pi 5 and Roborock LiDAR
Custom PCB designed in KiCad

Project Overview

This research project aims to build a low-cost autonomous floor-cleaning robot using ROS 2 for SLAM and navigation. A Raspberry Pi 5 runs ROS 2 Humble and handles high-level planning while an STM32 controls DC motors with encoder feedback.
